## Introduction

This portfolio website is built using [Astro](https://astro.build/), a modern static site generator. 

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/Srinath10X/portfolio.git
```
2. Navigate to the project directory:
```bash
cd portfolio
```
3. Install the dependencies:
```bash
npm install
```
## Usage
To start the development server, run:
```bash
npm run dev
```
Open your browser and go to `http://localhost:4321` to view your portfolio.
